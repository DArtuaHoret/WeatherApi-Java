# Weather API Java

## Jak uruchomić

### Opcja 1 – IntelliJ

1. Sklonuj repo:
   git clone https://github.com/DArtuaHoret/WeatherApi-Java
2. Otwórz w IntelliJ
3. Uruchom klasę `Main.java`

### Opcja 2 – Maven (terminal)

1. Wejdź do folderu projektu
2. Uruchom:
   mvn clean install
   mvn exec:java

## Wymagania

* Java 17+
* Maven
